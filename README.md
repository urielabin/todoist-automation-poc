# todoist-wdio-supertest-poc

Automated tests for todoist web app.

# Tools

- WebdriverIO
- Supertest
- Artillery
- ESLint
- Prettier
- Babel

# Test Execution

Run tests by command line:

- API

```
$ npm test
```

- Performance/Load

```
$ npm run test-performance
```

- Frontend

```
$ npm run test-frontend
```
